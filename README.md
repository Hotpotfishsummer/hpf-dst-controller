# 饥荒联机版 - 增强手柄控制 Mod

这是一个面向 Don't Starve Together 的手柄增强 Mod，提供虚拟光标、地图自动寻路、按钮组合和游戏内配置。

## 主要功能

- 自定义手柄按键组合
- 虚拟光标与全屏鼠标操作
- 地图选点后自动寻路
- 虫洞追踪与路径可视化
- 游戏内配置，无需重启

## 操作入口

- 键盘配置：`Ctrl+K`
- 手柄配置：`LB+RB+Y`
- 地图选点：打开地图后用虚拟光标点击目标，关闭地图后继续移动
