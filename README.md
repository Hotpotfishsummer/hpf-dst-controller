# 饥荒联机版 - 增强手柄控制 Mod

这是一个面向 Don't Starve Together 的手柄增强 Mod，提供虚拟光标、地图自动寻路、按键映射和游戏内配置界面。

## 主要功能

- 手柄按键组合可自定义
- 虚拟光标支持全屏鼠标操作
- 地图点击后自动规划路径并移动
- 虫洞编号追踪与地图标记
- 游戏内实时配置，无需重启

## 地图寻路

1. 打开地图
2. 开启虚拟光标并点击目标位置
3. 关闭地图后角色自动移动到目标点

## 安装

### Steam 创意工坊
订阅后直接启动游戏即可。

### 手动安装
将模组放到 `Documents/Klei/DoNotStarveTogether/mods/`，然后在游戏内启用。

## 配置入口

- 键盘：`Ctrl+K`
- 手柄：`LB+RB+Y`
