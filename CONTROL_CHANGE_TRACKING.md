# 操作变更追踪

本文档用于记录本 Mod 对游戏内默认操作做出的改写，以及 Mod 新增的操作入口。当前内容按键鼠与手柄分别列出，便于后续维护和对照。

## 1. 键鼠操作

| 场景 | 操作 | 功能 | 说明 |
|---|---|---|---|
| 全局 | Ctrl+K | 打开 Mod 配置界面 | 进入游戏内配置页 |
| 地图界面 | 鼠标左键点击地图 | 选择目标并开始自动寻路 | 地图打开后会自动启用虚拟光标 |
| 地图界面 | 鼠标滚轮 / 原生缩放操作 | 放大或缩小地图 | 缩放时会同步刷新路径点和虫洞标记 |
| 地图界面 | 鼠标拖动 / 原生平移操作 | 平移地图视图 | 路径和标记会跟随更新 |
| 虚拟光标模式 | 鼠标移动 | 控制光标移动 | 由虚拟光标系统接管 |
| 虚拟光标模式 | 鼠标左键 | 左键点击 | 用于界面确认、地图选点等 |
| 虚拟光标模式 | 鼠标右键 | 右键点击 | 用于取消、辅助交互等 |

## 2. 手柄操作

### 2.1 全局入口

| 物理按键 | 功能 | 说明 |
|---|---|---|
| LB + RB + Y | 打开 Mod 配置界面 | 手柄配置入口 |
| LB + RB + RT | 切换虚拟光标模式 | 手动开启 / 关闭虚拟光标 |
| LB | 修饰键 / 相机与界面辅助键 | 与其他按键组合时触发额外功能 |
| RB | 修饰键 / 指令轮盘辅助键 | 与其他按键组合时触发额外功能 |

### 2.2 地图界面改写

| 物理按键 | 功能 | 说明 |
|---|---|---|
| LB + 右摇杆上/下 | 地图缩放 | 对应放大 / 缩小地图 |
| LB + 右摇杆左/右 | 视角旋转 | 对应左转 / 右转视角 |
| LT | 左键点击 / 确认 | 虚拟光标模式下用于左键点击 |
| RT | 右键点击 / 取消 | 虚拟光标模式下用于右键点击 |
| 地图打开时自动启用虚拟光标 | 自动进入鼠标操作模式 | 关闭地图时自动退出 |
| 关闭地图 | 保持自动寻路继续执行 | 不会因为关图而停止移动 |

### 2.3 常用动作组合

#### LB 组合

| 组合 | 功能 | 说明 |
|---|---|---|
| LB + A | 攻击 | 对应普通攻击动作 |
| LB + Y | 检查 | 对应查看 / 检查动作 |
| LB + X | 当前保留为原生攻击相关逻辑 | 具体行为受原生控制与配置影响 |
| LB + B | 当前默认未绑定特殊动作 | 可在配置中重新定义 |
| LB + LT | 当前默认未绑定特殊动作 | 可在配置中重新定义 |
| LB + RT | 当前默认未绑定特殊动作 | 可在配置中重新定义 |

#### RB 组合

| 组合 | 功能 | 说明 |
|---|---|---|
| RB + B | 切换身体装备 | 循环身体栏装备 |
| RB + X | 切换手部装备 | 循环手部栏装备 |
| RB + Y | 切换头部装备 | 循环头部栏装备 |
| RB + A | 当前默认未绑定特殊动作 | 可在配置中重新定义 |
| RB + LT | 当前默认未绑定特殊动作 | 可在配置中重新定义 |
| RB + RT | 当前默认未绑定特殊动作 | 可在配置中重新定义 |

### 2.4 其他默认映射

| 物理按键 | 功能 | 说明 |
|---|---|---|
| LT | 制作栏 / 上下文主操作键 | 在不同界面中对应制作、菜单或虚拟光标左键 |
| RT | 物品栏 / 上下文副操作键 | 在不同界面中对应物品栏、菜单或虚拟光标右键 |
| RSTICK_RIGHT | 物品栏右移 / 切换值 | 兼容原生界面导航 |
| RSTICK_LEFT | 物品栏左移 / 切换值 | 兼容原生界面导航 |
| RSTICK_UP | 物品栏上移 | 兼容原生界面导航 |
| RSTICK_DOWN | 物品栏下移 | 兼容原生界面导航 |

## 3. 说明

- 地图缩放与视角旋转已被显式纳入本 Mod 的控制逻辑，后续若改动输入映射，应同步更新本文档。
- 虚拟光标在地图打开时会自动启用，地图关闭时自动退出。
- 地图选点后的自动寻路会持续执行，关闭地图不会中止移动；手动移动会取消自动寻路。
- 当前文档记录的是“现阶段已实现的操作”，后续增加新按键或改写默认行为时，建议直接在这里追加。
